# Security Policy

## Supported Versions

Fast Chapter is currently pre-1.0. Security fixes are applied to the latest `main` branch and latest release tag.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these private channels:
- GitHub Security Advisories for this repository (preferred)
- Direct maintainer contact: `alejandrogarcia2423@hotmail.com`

Please include:
- clear reproduction steps
- expected vs actual behavior
- potential impact
- environment details (OS, app version, Node version)

## Response Targets

- Initial acknowledgment: within 3 business days
- Triage/update: within 7 business days
- Fix timeline: based on severity and exploitability

## Disclosure

After a fix is available, coordinated public disclosure is encouraged with:
- affected versions
- mitigation guidance
- upgrade recommendation
